app <command> <option>

Usage:
  app --help                                  displays help for app usage
  app --init all                              creates folder structure and configuration files
  app init --mk                               creates just the folder structure
  app init --cat                              creates the default configuration files
  app init --help                             displays help for initialization processes
  app config --show                           displays the current configuration files in a list
  app config --reset                          resets the configuration files to default
  app config --set                            sets a specific configuration setting
  app config --help                           displays help for configuration processes
  app token --count                           displays the amount of tokens that have been created
  app token --new <username>                  generates a new token for the username inputted, and saves the token to the JSON file
  app token --update p <username> <phone>     updates the phone number for the specified user in the JSON file
  app token --update e <username> <email>     updates the email address for the specified user in the JSON file
  app token --fetch u <username>              fetches the token for the specified username
  app token --fetch e <email>                 fetches the token for the specified email
  app token --fetch p <phone>                 fetches the token for the specified phone number
  app token --help                            displays help for the token processes

Description:
  This is a command-line interface (CLI) application for various tasks. It provides functionality for initializing the app, configuring settings, and managing user tokens.

Commands:
  --help                 Displays the help message for the overall usage of the app.

Initialization Commands:
  init --mk              Creates only the folder structure for the app.
  init --cat             Creates the default configuration files.
  init --help            Displays help for the initialization processes.

Configuration Commands:
  config --show          Displays the current configuration files in a list.
  config --reset         Resets the configuration files to their default values.
  config --set           Sets a specific configuration setting.
  config --help          Displays help for the configuration processes.

Token Commands:
  token --count          Displays the amount of tokens that have been created.
  token --new <username> Generates a new token for the specified username and saves it to the JSON file.
  token --update p <username> <phone>     Updates the phone number for the specified user in the JSON file.
  token --update e <username> <email>     Updates the email address for the specified user in the JSON file.
  token --fetch u <username>              Fetches the token for the specified username.
  token --fetch e <email>                 Fetches the token for the specified email.
  token --fetch p <phone>                 Fetches the token for the specified phone number.
  token --help           Displays help for the token processes.
